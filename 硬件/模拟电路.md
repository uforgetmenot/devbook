# 模拟电路技术学习笔记

## 1. 模拟电路概述

### 1.1 定义与特点

模拟电路（Analog Circuit）是指处理连续变化信号的电子电路。与数字电路处理离散的高低电平不同，模拟电路处理的信号在时间上和幅度上都是连续变化的。模拟信号可以表示为时间的连续函数，如声音、温度、压力等自然界中的物理量。

模拟电路的核心特点是：
- 信号连续性：电压和电流随时间连续变化
- 幅度可变性：信号幅度可以在一定范围内任意取值
- 精度依赖性：电路性能受器件精度、温度、噪声等因素影响
- 非线性特性：许多模拟器件具有非线性特性

### 1.2 发展历程

模拟电路的发展可以追溯到19世纪末20世纪初：
- 1876年：贝尔发明电话，标志着电子通信的开始
- 1904年：弗莱明发明二极管，第一个电子器件
- 1906年：德福雷斯特发明三极管，开启电子放大时代
- 1947年：肖克利等人发明晶体管，革命性突破
- 1958年：杰克·基尔比发明集成电路
- 1970年代至今：模拟集成电路快速发展

### 1.3 应用领域

模拟电路在现代电子系统中应用广泛：
- 通信系统：射频放大器、混频器、调制解调器
- 音频处理：功放、滤波器、音效处理
- 传感器接口：信号调理、放大、滤波
- 电源管理：线性稳压器、开关电源控制器
- 测量仪器：示波器、频谱分析仪
- 生物医学：心电图、脑电图信号处理

## 2. 基础元器件与特性

### 2.1 电阻器

电阻器是模拟电路中最基本的被动器件之一，用于限制电流、分压和消耗功率。

#### 2.1.1 基本特性
- 欧姆定律：V = IR
- 功率消耗：P = I²R = V²/R
- 电阻温度系数：α = (R₂-R₁)/(R₁×(T₂-T₁))

#### 2.1.2 电阻器类型
1. **碳膜电阻**：
   - 特点：成本低、噪声大、精度一般
   - 应用：一般电路中的限流、分压

2. **金属膜电阻**：
   - 特点：精度高、温度系数小、噪声低
   - 应用：精密测量、信号调理电路

3. **线绕电阻**：
   - 特点：功率大、精度高、电感大
   - 应用：功率电路、精密分压

4. **薄膜电阻**：
   - 特点：精度极高、温度系数极小
   - 应用：精密仪器、标准电阻

#### 2.1.3 电阻器选型要点
- 阻值选择：根据电路需求选择标准阻值
- 功率等级：确保实际功耗不超过额定功率
- 精度要求：根据电路精度要求选择容差
- 温度系数：精密应用需考虑温度漂移
- 频率特性：高频应用需考虑寄生电感和电容

### 2.2 电容器

电容器是存储电荷的器件，具有隔直通交的特性，在滤波、耦合、旁路等应用中发挥重要作用。

#### 2.2.1 基本特性
- 电容定义：C = Q/V
- 电流电压关系：i = C(dv/dt)
- 容抗：Xc = 1/(2πfC)
- 储能：W = ½CV²

#### 2.2.2 电容器类型
1. **陶瓷电容器**：
   - Class 1（NP0/C0G）：温度系数稳定，精度高，容量小
   - Class 2（X7R、Y5V）：容量大，温度系数变化大

2. **电解电容器**：
   - 铝电解：容量大，有极性，ESR较高
   - 钽电解：体积小，ESR低，成本高

3. **薄膜电容器**：
   - 聚酯、聚丙烯：损耗低，稳定性好
   - 应用：高频电路、滤波电路

#### 2.2.3 电容器选型要点
- 容量选择：根据电路需求选择合适容量
- 耐压选择：工作电压应小于额定电压的80%
- 介质类型：根据频率、温度、精度要求选择
- ESR和ESL：高频应用需考虑寄生参数
- 温度特性：考虑工作温度范围内的性能变化

### 2.3 电感器

电感器是利用电磁感应原理工作的器件，对交流信号呈现阻抗，具有储能和滤波功能。

#### 2.3.1 基本特性
- 电感定义：L = NΦ/I
- 电压电流关系：v = L(di/dt)
- 感抗：XL = 2πfL
- 储能：W = ½LI²

#### 2.3.2 电感器类型
1. **绕线电感**：
   - 功率电感：大电流，低DCR
   - 射频电感：高Q值，小尺寸

2. **铁氧体磁芯电感**：
   - 高电感值，适合低频应用
   - 饱和电流有限

3. **空心电感**：
   - 线性度好，Q值高
   - 电感值小，适合高频

#### 2.3.3 电感器选型要点
- 电感值：根据电路需求选择
- 饱和电流：确保工作电流小于饱和电流
- DCR（直流电阻）：影响效率和发热
- Q值：高频应用需考虑品质因数
- 封装尺寸：考虑PCB布局和散热

### 2.4 半导体器件

#### 2.4.1 二极管
二极管是最简单的半导体器件，具有单向导电特性。

**基本特性**：
- 正向导通：当正向电压超过阈值电压时导通
- 反向截止：反向电压下呈现高阻态
- 伏安特性：I = Is(e^(qV/nkT)-1)

**主要类型**：
1. **硅二极管**：正向压降约0.7V，耐高温
2. **锗二极管**：正向压降约0.3V，导通电压低
3. **肖特基二极管**：正向压降低，开关速度快
4. **齐纳二极管**：反向击穿特性，用于稳压
5. **变容二极管**：结电容随反向电压变化

#### 2.4.2 双极结型晶体管（BJT）
BJT是电流控制器件，有NPN和PNP两种类型。

**工作原理**：
- 共发射极配置：基极电流控制集电极电流
- 电流关系：Ic = βIb，Ie = Ic + Ib
- 放大条件：发射结正偏，集电结反偏

**特性参数**：
- 电流增益β：β = Ic/Ib
- 输入阻抗：中等，约1kΩ~5kΩ
- 输出阻抗：较高，约50kΩ~100kΩ
- 频率特性：截止频率fT

#### 2.4.3 场效应晶体管（FET）
FET是电压控制器件，分为JFET和MOSFET两大类。

**MOSFET特性**：
- 输入阻抗极高：10^9~10^12Ω
- 驱动功率小
- 热稳定性好
- 静电敏感

**工作区域**：
- 截止区：Vgs < Vth
- 线性区：Vgs > Vth，Vds < Vgs - Vth
- 饱和区：Vgs > Vth，Vds ≥ Vgs - Vth

## 3. 模拟电路分析方法

### 3.1 直流分析

直流分析用于确定电路的静态工作点（Q点），是模拟电路设计的基础。

#### 3.1.1 基本概念
- 静态工作点：晶体管在无信号输入时的电压电流值
- 直流负载线：表示电路中电压电流关系的直线
- 工作区域：放大区、饱和区、截止区

#### 3.1.2 分析步骤
1. **确定直流等效电路**：将电容视为开路，电感视为短路
2. **应用基尔霍夫定律**：列出节点电压方程或回路电流方程
3. **求解静态工作点**：确定各节点的直流电压和电流
4. **验证工作状态**：检查晶体管是否工作在预期区域

#### 3.1.3 实例分析
考虑共射极放大电路：
```
Vcc
 |
 Rc
 |
 Cc1 ---+---> Vout
       |
       +--- Collector
       |
       +--- Base (通过Rb接Vin)
       |
       +--- Emitter (接地)
       |
 Re
 |
GND
```

分析步骤：
1. 确定基极电流：Ib = (Vcc - Vbe) / (Rb + (β+1)Re)
2. 计算集电极电流：Ic = βIb
3. 计算集电极-发射极电压：Vce = Vcc - IcRc - IeRe
4. 验证工作状态：确保Vce > Vcesat，Vbe > 0.6V

### 3.2 交流分析

交流分析用于研究电路对小信号的响应特性。

#### 3.2.1 小信号模型
- BJT小信号模型：h参数模型、混合π模型
- FET小信号模型：跨导模型、T模型
- 模型参数：跨导gm、输入电阻rπ、输出电阻ro

#### 3.2.2 频率响应分析
- 低频响应：主要受耦合电容和旁路电容影响
- 中频响应：电路的正常放大工作频段
- 高频响应：受器件寄生参数和分布参数影响

#### 3.2.3 传递函数
电路的传递函数描述了输出与输入的复频域关系：
H(s) = Vout(s) / Vin(s)

其中s = σ + jω，用于分析电路的频率响应和稳定性。

### 3.3 瞬态分析

瞬态分析研究电路在时间域内的响应特性。

#### 3.3.1 RC电路瞬态响应
- 充电过程：Vc(t) = V(1 - e^(-t/RC))
- 放电过程：Vc(t) = V0e^(-t/RC))
- 时间常数：τ = RC

#### 3.3.2 RL电路瞬态响应
- 电流建立：i(t) = (V/R)(1 - e^(-Rt/L))
- 电流衰减：i(t) = I0e^(-Rt/L)
- 时间常数：τ = L/R

#### 3.3.3 RLC电路瞬态响应
- 欠阻尼：振荡衰减
- 临界阻尼：最快无振荡响应
- 过阻尼：缓慢无振荡响应

## 4. 放大器电路设计

### 4.1 放大器基本概念

放大器是模拟电路的核心，用于增强信号的幅度、功率或电流。

#### 4.1.1 放大器分类
1. **按器件分类**：
   - BJT放大器
   - FET放大器
   - 真空管放大器
   - 运算放大器

2. **按配置分类**：
   - 共射极/共源极：电压放大
   - 共集电极/共漏极：电流缓冲
   - 共基极/共栅极：高频应用

3. **按频率分类**：
   - 低频放大器：音频应用
   - 中频放大器：通信应用
   - 高频放大器：射频应用

#### 4.1.2 放大器参数
- 电压增益：Av = Vout/Vin
- 电流增益：Ai = Iout/Iin
- 功率增益：Ap = Pout/Pin
- 输入阻抗：Zin = Vin/Iin
- 输出阻抗：Zout = Vout/Iout
- 带宽：放大器能正常工作的频率范围
- 失真度：输出信号与输入信号的差异

### 4.2 BJT放大器设计

#### 4.2.1 共射极放大器
共射极放大器是最常用的放大器配置，具有电压和电流放大功能。

**电路配置**：
- 输入信号加在基极
- 输出信号取自集电极
- 发射极接地（可能通过电阻接地）

**设计要点**：
1. **静态工作点设置**：
   - 选择合适的基极偏置电阻
   - 确保Q点在负载线中点附近
   - 考虑温度稳定性

2. **增益计算**：
   - 电压增益：Av = -gmRc || RL
   - 其中gm = Ic/Vt ≈ 40Ic (mA/V)

3. **输入输出阻抗**：
   - 输入阻抗：Zin ≈ βre，其中re = 26mV/Ic(mA)
   - 输出阻抗：Zout ≈ Rc

#### 4.2.2 共集电极放大器（射极跟随器）
射极跟随器具有高输入阻抗、低输出阻抗的特性，常用作缓冲器。

**电路特点**：
- 电压增益略小于1
- 电流增益高
- 输入阻抗高
- 输出阻抗低

**应用**：阻抗匹配、驱动低阻负载

#### 4.2.3 共基极放大器
共基极放大器具有高频特性好、电压增益高、电流增益小于1的特点。

**应用**：高频放大、电流缓冲

### 4.3 FET放大器设计

#### 4.3.1 共源极放大器
共源极放大器是FET放大器中最常用的配置。

**电路特点**：
- 电压增益：Av = -gm(rd || Rd)
- 输入阻抗：极高，约兆欧级别
- 输出阻抗：取决于负载电阻

**偏置方法**：
1. **自偏置**：利用源极电阻产生负反馈
2. **分压偏置**：使用电阻分压稳定栅极电压
3. **电流源偏置**：使用电流源提供稳定偏置

#### 4.3.2 共漏极放大器（源极跟随器）
源极跟随器具有高输入阻抗、低输出阻抗的特点。

**特点**：
- 电压增益小于1但接近1
- 电流增益高
- 输入阻抗极高
- 输出阻抗较低

### 4.4 运算放大器应用

运算放大器（Op-Amp）是模拟电路中的重要集成器件，具有高增益、高输入阻抗、低输出阻抗的特点。

#### 4.4.1 理想运放特性
- 开环增益无穷大：Aol → ∞
- 输入阻抗无穷大：Zin → ∞
- 输出阻抗为零：Zout = 0
- 带宽无穷大
- 共模抑制比无穷大

#### 4.4.2 基本运放电路

1. **反相放大器**：
   ```
   Vin --- Rin ---+--- Vout
                  |
                 ---
                 | | Rf
                 | |
                 ---
                  |
                 GND
   ```
   电压增益：Av = -Rf/Rin

2. **同相放大器**：
   ```
   Vin ---+---------- Vout
          |
         ---
         | | R1
         | |
         ---
          |
         GND
   ```
   电压增益：Av = 1 + Rf/R1

3. **电压跟随器**：
   - 增益为1
   - 输入阻抗极高
   - 输出阻抗极低

4. **加法器**：
   多个输入信号通过不同电阻连接到反相输入端

5. **差分放大器**：
   放大两个输入信号的差值

#### 4.4.3 实际运放考虑
- 输入偏置电流
- 输入失调电压
- 压摆率
- 增益带宽积
- 电源抑制比

## 5. 滤波器设计

滤波器是从输入信号中选择特定频率成分的电路。

### 5.1 滤波器分类

#### 5.1.1 按频率特性分类
1. **低通滤波器（LPF）**：通过低频，抑制高频
2. **高通滤波器（HPF）**：通过高频，抑制低频
3. **带通滤波器（BPF）**：通过特定频段
4. **带阻滤波器（BSF）**：抑制特定频段
5. **全通滤波器**：相位变化，幅度不变

#### 5.1.2 按实现方式分类
1. **无源滤波器**：使用R、L、C元件
2. **有源滤波器**：使用运放等有源器件
3. **开关电容滤波器**：使用开关电容技术

### 5.2 无源滤波器设计

#### 5.2.1 一阶RC低通滤波器
电路由电阻和电容串联组成。

**传递函数**：
H(jω) = 1/(1 + jωRC)

**截止频率**：
fc = 1/(2πRC)

**特点**：
- 低频增益为1
- 高频衰减率为-20dB/十倍频
- 相位滞后从0°到-90°

#### 5.2.2 一阶RC高通滤波器
电容和电阻位置互换。

**传递函数**：
H(jω) = jωRC/(1 + jωRC)

**特点**：
- 高频增益为1
- 低频衰减率为-20dB/十倍频
- 相位超前从90°到0°

### 5.3 有源滤波器设计

有源滤波器克服了无源滤波器的增益损失问题，具有更好的性能。

#### 5.3.1 Sallen-Key滤波器
Sallen-Key是二阶有源滤波器的常用拓扑。

**电路结构**：
- 使用单个运放
- 两个电阻，两个电容
- 反馈网络

**传递函数**：
H(s) = K/(s² + (ω₀/Q)s + ω₀²)

其中K为增益，ω₀为特征频率，Q为品质因数。

#### 5.3.2 多反馈滤波器
多反馈拓扑提供更好的高频性能。

**特点**：
- 单位增益稳定
- 增益与Q值相互独立
- 适用于高Q值应用

### 5.4 滤波器设计参数

#### 5.4.1 品质因数（Q值）
Q值描述了滤波器的频率选择性：
- Q = 1/(2ζ)，其中ζ为阻尼比
- 高Q值：频率选择性好，但带宽窄
- 低Q值：带宽宽，但选择性差

#### 5.4.2 滤波器逼近类型
1. **巴特沃斯滤波器**：
   - 幅频响应单调
   - 相位响应线性度好
   - 过渡带较宽

2. **切比雪夫滤波器**：
   - 过渡带陡峭
   - 通带内有纹波
   - 相位非线性

3. **贝塞尔滤波器**：
   - 群延迟恒定
   - 相位线性度最佳
   - 过渡带较缓

## 6. 振荡器电路

振荡器是产生周期性信号的电路，在时钟、载波、测试信号等应用中不可或缺。

### 6.1 振荡器原理

振荡器基于正反馈原理工作，需要满足巴克豪森准则：
1. 环路增益：|Aβ| ≥ 1
2. 相位条件：∠Aβ = 0°或360°的整数倍

### 6.2 LC振荡器

LC振荡器利用LC谐振电路产生正弦波。

#### 6.2.1 哈特利振荡器
使用两个电感和一个电容的振荡电路。

**特点**：
- 频率可调
- 结构简单
- 频率稳定性一般

#### 6.2.2 科利特振荡器
使用一个电感和两个电容的振荡电路。

**特点**：
- 频率稳定
- 输出纯度高
- 频率调节方便

#### 6.2.3 克拉普振荡器
改进的科利特振荡器，增加了额外电容。

**特点**：
- 频率调节范围大
- 稳定性好
- 适用于VCO设计

### 6.3 RC振荡器

RC振荡器使用电阻和电容网络产生振荡。

#### 6.3.1 相移振荡器
使用RC网络产生180°相移。

**电路组成**：
- 三级RC网络：每级60°相移
- 反相放大器：提供180°相移
- 总相移：360°

**频率计算**：
f = 1/(2πRC√6)

#### 6.3.2 文氏桥振荡器
使用文氏桥网络作为选频网络。

**特点**：
- 频率稳定
- 失真小
- 适用于音频应用

**频率计算**：
f = 1/(2πRC)

### 6.4 晶体振荡器

晶体振荡器利用石英晶体的压电效应产生高稳定频率。

#### 6.4.1 晶体特性
- 串联谐振频率：fs = 1/(2π√LCs)
- 并联谐振频率：fp = 1/(2π√LCp)
- 频率稳定度：可达10⁻⁶~10⁻⁹

#### 6.4.2 晶体振荡器电路
1. **皮尔斯振荡器**：最常用的CMOS晶体振荡器
2. **CMOS反相器振荡器**：简单易实现
3. **双极型晶体振荡器**：适用于低功耗应用

## 7. 电源电路设计

电源电路为电子系统提供稳定的工作电压。

### 7.1 线性稳压器

线性稳压器通过调整内部器件的导通程度来调节输出电压。

#### 7.1.1 基本原理
- 串联调节元件：BJT或MOSFET
- 误差放大器：比较反馈电压与基准电压
- 基准源：提供稳定参考电压
- 过载保护：防止器件损坏

#### 7.1.2 三端线性稳压器
- 固定输出：78xx系列，79xx系列
- 可调输出：LM317，LM337

**使用要点**：
- 输入输出压差足够
- 足够的散热措施
- 输入输出电容配置

#### 7.1.3 低压差稳压器（LDO）
LDO提供极小的输入输出压差。

**特点**：
- 压差小：通常<100mV
- 噪声低：适合模拟电路供电
- 静态电流小：适合电池供电

### 7.2 开关电源

开关电源通过高频开关控制能量传输，效率高。

#### 7.2.1 基本拓扑
1. **Buck（降压）变换器**：
   - 输出电压低于输入电压
   - 效率高，应用广泛

2. **Boost（升压）变换器**：
   - 输出电压高于输入电压
   - 用于电池升压应用

3. **Buck-Boost变换器**：
   - 输出电压可高于或低于输入电压
   - 输出极性相反

4. **Flyback变换器**：
   - 隔离型变换器
   - 适合多路输出

#### 7.2.2 控制方式
1. **PWM控制**：脉宽调制，固定频率
2. **PFM控制**：脉冲频率调制，可变频率
3. **混合控制**：结合PWM和PFM优点

### 7.3 电源设计考虑

#### 7.3.1 纹波和噪声
- 纹波：电源输出的交流成分
- 噪声：高频干扰成分
- 抑制方法：滤波、稳压、屏蔽

#### 7.3.2 瞬态响应
- 负载瞬变时的输出变化
- 响应时间要求
- 补偿网络设计

#### 7.3.3 效率优化
- 选择合适的拓扑
- 优化开关频率
- 优化器件选择

## 8. 非线性电路

非线性电路的输出与输入不成线性关系。

### 8.1 二极管应用电路

#### 8.1.1 整流电路
1. **半波整流**：
   - 电路简单
   - 效率低
   - 纹波大

2. **全波整流**：
   - 中心抽头变压器
   - 效率较高

3. **桥式整流**：
   - 不需要中心抽头
   - 应用广泛
   - 效率高

#### 8.1.2 限幅电路
- 串联限幅器：利用二极管正向导通
- 并联限幅器：利用二极管反向击穿
- 双向限幅：双向保护

#### 8.1.3 钳位电路
- 正钳位：将信号底部钳制到特定电平
- 负钳位：将信号顶部钳制到特定电平
- 应用：信号恢复、直流恢复

### 8.2 比较器电路

比较器用于比较两个电压的大小。

#### 8.2.1 基本比较器
- 开环运放作为比较器
- 高增益，快速响应
- 输出为数字电平

#### 8.2.2 窗口比较器
- 两个阈值：上限和下限
- 三态输出：高、低、中
- 应用：电压监控

#### 8.2.3 迟滞比较器
- 引入正反馈
- 避免噪声引起的振荡
- 两个阈值：上阈值和下阈值

### 8.3 乘法器和调制

#### 8.3.1 模拟乘法器
- 四象限乘法器
- 应用：调制、解调、混频
- 电路实现：吉尔伯特单元

#### 8.3.2 调制电路
- 调幅（AM）：载波幅度随调制信号变化
- 调频（FM）：载波频率随调制信号变化
- 调相（PM）：载波相位随调制信号变化

## 9. 频率响应与稳定性

### 9.1 频率响应分析

频率响应描述电路对不同频率信号的响应特性。

#### 9.1.1 幅频特性
- 低频段：受耦合电容影响
- 中频段：电路正常工作频段
- 高频段：受寄生参数影响

#### 9.1.2 相频特性
- 相位变化：影响信号完整性
- 群延迟：不同频率的延迟差异
- 相位裕度：稳定性指标

### 9.2 稳定性分析

稳定性是放大器正常工作的基本要求。

#### 9.2.1 稳定性准则
- 奈奎斯特稳定判据
- 波特图稳定性判据
- 相位裕度：>45°
- 增益裕度：>6dB

#### 9.2.2 补偿技术
1. **主极点补偿**：
   - 降低增益带宽积
   - 稳定性好
   - 速度慢

2. **超前补偿**：
   - 增加相位裕度
   - 保持带宽
   - 设计复杂

3. **滞后补偿**：
   - 改善低频特性
   - 降低高频增益

### 9.3 反馈理论

反馈是改善电路性能的重要手段。

#### 9.3.1 反馈类型
- 电压串联反馈：提高输入阻抗，降低输出阻抗
- 电压并联反馈：降低输入阻抗，降低输出阻抗
- 电流串联反馈：提高输入阻抗，提高输出阻抗
- 电流并联反馈：降低输入阻抗，提高输出阻抗

#### 9.3.2 反馈方程
闭环增益：Acl = Aol/(1 + Aolβ)

其中Aol为开环增益，β为反馈系数。

## 10. 噪声分析

噪声是影响模拟电路性能的重要因素。

### 10.1 噪声类型

#### 10.1.1 电阻热噪声
- 产生机理：电阻内载流子热运动
- 噪声功率：Pn = 4kTRB
- 频率特性：白噪声，与频率无关

#### 10.1.2 半导体噪声
1. **散粒噪声**：载流子随机通过势垒
2. **闪烁噪声（1/f噪声）**：低频噪声
3. **爆米花噪声**：离散噪声

#### 10.1.3 1/f噪声
- 与频率成反比
- 低频影响大
- 与器件工艺相关

### 10.2 噪声分析方法

#### 10.2.1 噪声等效电路
- 输入噪声源等效
- 输出噪声计算
- 噪声系数定义

#### 10.2.2 信噪比（SNR）
SNR = 信号功率/噪声功率

影响因素：
- 信号幅度
- 噪声水平
- 带宽

### 10.3 噪声抑制

#### 10.3.1 电路级抑制
- 选择低噪声器件
- 优化偏置电流
- 使用噪声匹配技术

#### 10.3.2 系统级抑制
- 屏蔽和接地
- 滤波技术
- 差分技术

## 11. 实战案例

### 11.1 音频放大器设计

设计一个高质量的音频功率放大器，要求输出功率10W，频率响应20Hz-20kHz。

#### 11.1.1 需求分析
- 功率要求：10W输出到8Ω负载
- 频率范围：20Hz-20kHz（音频范围）
- 失真度：THD < 1%
- 信噪比：>80dB

#### 11.1.2 方案设计
采用B类推挽放大器结构：
- 输入级：差分放大器
- 驱动级：电压放大
- 输出级：互补对称推挽
- 反馈：负反馈改善线性度

#### 11.1.3 电路实现
1. **偏置电路**：
   - 二极管串提供Vbe倍数偏置
   - 温度补偿

2. **输出级设计**：
   - 互补对称输出
   - 保护电路：过流、过温

3. **反馈网络**：
   - 电压串联负反馈
   - 频率补偿

#### 11.1.4 性能优化
- 交越失真消除
- 稳定性补偿
- 散热设计

### 11.2 传感器信号调理电路

设计一个温度传感器信号调理电路，传感器输出0-100mV，对应0-100°C。

#### 11.2.1 需求分析
- 输入：0-100mV，对应0-100°C
- 输出：0-5V，对应0-100°C
- 精度：±0.5°C
- 线性度：良好

#### 11.2.2 电路设计
1. **前置放大**：
   - 仪表放大器：高输入阻抗，高共模抑制比
   - 增益：50倍

2. **滤波电路**：
   - 低通滤波：抑制高频噪声
   - 截止频率：100Hz

3. **电平移位**：
   - 消除传感器偏移
   - 输出0-5V

#### 11.2.3 器件选择
- 仪表放大器：AD620，INA128
- 精密运放：OP07，OP27
- 基准电压：TL431，REF02

### 11.3 射频放大器设计

设计一个800MHz的低噪声放大器，用于接收机前端。

#### 11.3.1 技术指标
- 频率：800MHz
- 增益：>15dB
- 噪声系数：<2dB
- 输入匹配：S11 <-10dB
- 输出匹配：S22 <-10dB

#### 11.3.2 晶体管选择
- 选择高频FET：BFP640，SST127
- 查看S参数
- 稳定性分析

#### 11.3.3 匹配网络设计
1. **输入匹配**：
   - 使用Smith圆图
   - L型匹配网络

2. **输出匹配**：
   - 负载线匹配
   - 考虑功率传输

#### 11.3.4 稳定性考虑
- 稳定性因子K > 1
- Δ < 1
- 增益稳定
