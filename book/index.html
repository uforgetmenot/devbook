PCB制板 (PCB Fabrication)

1. 基材选择 (Substrate Selection)

1.1 FR-4材料

技术原理：

  • 玻璃纤维布加环氧树脂复合材料
  • 优异的机械强度和电气绝缘性能
  • 工作温度范围：-55°C ~ +130°C

关键参数：

  • 介电常数 (Dk)：4.2-4.7 @ 1MHz
  • 介电损耗 (Df)：0.018-0.025 @ 1MHz
  • 玻璃转换温度 (Tg)：130°C-180°C
  • 分解温度 (Td)：>300°C

应用场景：

  • 一般消费电子产品
  • 工业控制设备
  • 通信设备（<1GHz）

1.2 Rogers高频材料

技术原理：

  • 聚四氟乙烯(PTFE)或改性环氧树脂
  • 低介电常数和损耗
  • 优异的频率稳定性

常用型号及参数：

  • RO4003C: Dk=3.38±0.05, Df=0.0027 @ 10GHz
  • RO4350B: Dk=3.48±0.05, Df=0.0037 @ 10GHz
  • RT/duroid 5880: Dk=2.20±0.02, Df=0.0009 @ 10GHz

应用场景：

  • 射频/微波电路
  • 5G通信设备
  • 雷达系统
  • 航空航天设备

质量控制要点：

  • 介电常数均匀性检测
  • 厚度公差控制 (±7%)
  • 表面粗糙度测量
  • 含水率控制 (<0.02%)

2. 内层图形转移 (Inner Layer Patterning)

2.1 覆铜板处理 (CCL Preparation)

工艺流程：

  1. 来料检验 → 2. 表面清洁 → 3. 粗化处理 → 4. 干燥

关键参数：

  • 表面清洁：碱性清洗剂，pH 10-12
  • 微蚀刻深度：0.5-1.5μm
  • 表面粗糙度：Ra 0.5-1.2μm
  • 残留水分：<0.02%

2.2 干膜光刻胶工艺 (Dry Film Photoresist)

技术原理：

  • 负性光刻胶：曝光区域聚合交联，不被显影液溶解
  • 正性光刻胶：曝光区域分解，被显影液溶解

工艺参数：

  • 贴膜温度：100-120°C
  • 贴膜压力：0.2-0.4MPa
  • 贴膜速度：1.5-3.0m/min
  • 膜厚控制：25-75μm

质量控制：

  • 膜厚均匀性：±10%
  • 贴膜气泡率：<1%
  • 边缘密合度检查

2.3 曝光显影 (Exposure & Development)

曝光工艺：

  • 光源：UV灯 (350-450nm)
  • 曝光能量：80-150mJ/cm²
  • 曝光间隙：25-75μm
  • 分辨率：≤50μm线宽/间距

显影工艺：

  • 显影液：1%碳酸钠水溶液
  • 显影温度：30±2°C
  • 显影时间：60-120秒
  • 显影压力：0.1-0.2MPa

检验标准：

  • 线宽精度：±10%
  • 图形完整性：100%
  • 残胶检查：无残留
  • 尺寸偏差：±0.05mm

2.4 蚀刻工艺 (Etching)

酸性蚀刻：

  • 蚀刻液：CuCl₂ + HCl
  • 蚀刻温度：45-55°C
  • 蚀刻速率：25-35μm/min
  • 铜离子浓度：120-160g/L

碱性蚀刻：

  • 蚀刻液：CuCl₂ + NH₄Cl + NH₃·H₂O
  • 蚀刻温度：45-50°C
  • 蚀刻速率：20-30μm/min
  • pH值：8.0-8.5

侧蚀控制：

  • 侧蚀系数：<1:3
  • 蚀刻因子：>2.5
  • 残铜检查：<5μm

常见缺陷及解决方案：

  • 过蚀刻 → 降低温度/浓度，减少时间
  • 欠蚀刻 → 增加温度/浓度，延长时间
  • 侧蚀严重 → 优化蚀刻液配方，控制喷淋压力

3. 层压 (Lamination)

3.1 叠层结构设计

材料构成：

  • 内层板：已图形化的铜箔基板
  • 预浸料 (Prepreg)：半固化片，提供绝缘和粘结
  • 外层铜箔：18μm, 35μm, 70μm厚度选择

叠层计算：

  • 阻抗控制要求
  • 厚度公差分配
  • 压合后回弹补偿

3.2 层压工艺参数

温度曲线：

  1. 升温阶段：1-3°C/min → 170°C
  2. 固化阶段：170-180°C，保持60-90min
  3. 冷却阶段：2-3°C/min → 60°C

压力控制：

  • 初始压力：0.1-0.2MPa
  • 固化压力：1.5-2.5MPa
  • 压力保持至冷却<60°C

真空控制：

  • 真空度：-0.08 ~ -0.10MPa
  • 真空保持至胶化点

质量控制指标：

  • 厚度公差：±10%
  • 翘曲度：<0.75%
  • 分层检查：无分层气泡
  • 阻抗测试：±10%目标值

3.3 常见问题及对策

分层现象：

  • 原因：预浸料含胶量不足，压合参数不当
  • 解决：调整胶含量35-45%，优化温度曲线

厚度不均：

  • 原因：压合压力分布不均，垫板平整度差
  • 解决：校正压机平行度，更换垫板

翘曲变形：

  • 原因：铜厚不对称，应力释放不当
  • 解决：对称设计，控制冷却速率

4. 钻孔 (Drilling)

4.1 机械钻孔 (Mechanical Drilling)

设备配置：

  • 主轴转速：40,000-120,000 RPM
  • 进给速度：1000-8000 mm/min
  • 钻头规格：0.1-6.35mm
  • 定位精度：±0.025mm

钻头选择：

  • 硬质合金钻头：φ0.1-0.5mm
  • 高速钢钻头：φ0.5-6.35mm
  • 涂层钻头：TiN, TiAlN, DLC涂层

钻孔质量控制：

  • 孔径公差：±0.05mm
  • 孔壁粗糙度：Ra<3.2μm
  • 钻污控制：<25μm
  • 毛刺高度：<25μm

4.2 激光钻孔 (Laser Drilling)

技术原理：

  • CO₂激光：波长10.6μm，适用于有机材料
  • UV激光：波长355nm，适用于精密钻孔
  • 脉冲激光：脉冲宽度ns-fs级别

工艺参数：

  • 功率密度：10⁶-10⁸ W/cm²
  • 脉冲频率：1-50kHz
  • 扫描速度：100-1000mm/s
  • 焦距控制：±0.01mm

Microvia制作：

  • 孔径范围：50-150μm
  • 深径比：1:1 ~ 1:10
  • 锥度控制：5-15°
  • 精度要求：±10μm

4.3 通孔/盲孔/埋孔工艺

通孔 (Through Hole)：

  • 贯穿所有层
  • 最大深径比：1:12
  • 常用于插件元器件

盲孔 (Blind Via)：

  • 连接外层与内层
  • 深径比限制：1:8
  • 激光钻孔+电镀

埋孔 (Buried Via)：

  • 连接内层与内层
  • 在层压前制作
  • 需要特殊叠层工艺

质量检验方法：

  • 显微镜检查：孔径、锥度、毛刺
  • 切片分析：孔壁质量、镀层厚度
  • 电性测试：导通性、绝缘性

5. 孔金属化 (Plating)

5.1 化学沉铜 (Electroless Copper)

工艺原理：

  • 催化还原反应：Cu²⁺ + 2e⁻ → Cu
  • 自催化过程：铜层促进进一步沉积
  • 厚度控制：0.2-1.0μm

工艺流程：

  1. 膨胀 → 2. 高锰酸钾粗化 → 3. 中和 → 4. 预浸 → 5. 活化 → 6. 还原 → 7. 化学沉铜

关键参数：

  • 膨胀温度：80-90°C，时间：10-15min
  • 粗化浓度：60-80g/L KMnO₄
  • 活化液：PdCl₂ 0.1-0.3g/L
  • 沉铜速度：1.5-3.0μm/h

质量控制：

  • 沉铜厚度：0.3-0.8μm
  • 覆盖率：100%
  • 结合力：>1.2N/mm
  • 孔壁连续性：无断点

5.2 电镀铜工艺 (Electroplating Copper)

电镀原理：

  • 阳极：磷铜球 (含磷0.04-0.08%)
  • 阴极：PCB板
  • 电解液：CuSO₄·5H₂O + H₂SO₄

工艺参数：

  • 硫酸铜浓度：200-250g/L
  • 硫酸浓度：50-60g/L
  • 电流密度：2-4A/dm²
  • 镀液温度：22-28°C
  • 阴极移动速度：10-15m/min

添加剂管理：

  • 整平剂：控制表面平整度
  • 光亮剂：提高表面光泽
  • 抑制剂：防止烧焦现象
  • 载体：传输有机添加剂

镀层要求：

  • 孔壁镀铜厚度：20-25μm
  • 表面镀铜厚度：18-25μm
  • 厚度均匀性：±20%
  • 结合强度：>1.8N/mm

常见缺陷处理：

  • 镀层粗糙 → 降低电流密度，增加添加剂
  • 镀层发暗 → 检查阳极状态，调整pH值
  • 厚度不均 → 优化挂具设计，改善电流分布

6. 外层图形转移 (Outer Layer Patterning)

6.1 电镀铜锡工艺

工艺目的：

  • 铜层：增加导体厚度
  • 锡层：保护铜不被蚀刻

电镀铜参数：

  • 电流密度：2-3A/dm²
  • 镀液温度：22-26°C
  • 镀层厚度：15-25μm
  • 镀速：15-20μm/h

电镀锡参数：

  • 锡酸钠浓度：40-60g/L
  • 电流密度：1-2A/dm²
  • 镀液温度：20-25°C
  • 镀层厚度：2.5-7.5μm

6.2 图形电镀质量控制

前处理要求：

  • 表面清洁度：无油污、氧化
  • 微蚀刻深度：0.5-1.5μm
  • 活化处理：稀硫酸浸泡

镀层检验：

  • 厚度测量：X-ray荧光测厚仪
  • 结合力测试：胶带法、弯曲法
  • 外观检查：光泽度、针孔、粗糙度

6.3 蚀刻去膜

蚀刻工艺：

  • 蚀刻液：碱性氯化铜
  • 蚀刻温度：50±3°C
  • 蚀刻时间：根据铜厚调整
  • 蚀刻速率：25-35μm/min

去锡工艺：

  • 硝酸去锡：HNO₃浓度15-20%
  • 去锡温度：30-40°C
  • 去锡时间：30-60秒

去膜工艺：

  • 氢氧化钠溶液：浓度3-5%
  • 去膜温度：50-60°C
  • 去膜时间：2-5分钟

7. 阻焊层 (Solder Mask)

7.1 阻焊油墨类型

感光型阻焊油墨：

  • 液态感光油墨：LPI (Liquid Photo Imageable)
  • 干膜感光油墨：DFSM (Dry Film Solder Mask)

热固化型阻焊油墨：

  • 丝网印刷应用
  • 成本较低，精度有限

7.2 LPI工艺流程

前处理：

  1. 除油清洁 → 2. 微蚀刻 → 3. 水洗干燥 → 4. 黑化处理

油墨涂布：

  • 涂布方式：帘式涂布、辊涂
  • 涂布厚度：15-30μm (湿膜)
  • 固化后厚度：10-25μm
  • 涂布均匀性：±3μm

曝光显影：

  • 曝光能量：100-300mJ/cm²
  • 曝光分辨率：75μm开窗
  • 显影液：1%碳酸钠
  • 显影温度：30±2°C

最终固化：

  • 固化温度：150±5°C
  • 固化时间：30-60分钟
  • 固化后硬度：6H以上

7.3 质量控制标准

外观要求：

  • 颜色均匀一致
  • 表面光滑无气泡
  • 边缘整齐无毛刺
  • 开窗尺寸准确

性能测试：

  • 附着力：≥5B (百格法)
  • 耐温性：288°C，10秒无起泡
  • 阻燃性：符合UL94 V-0级
  • 绝缘电阻：>10¹²Ω

常见缺陷及原因：

  • 开窗偏大 → 曝光过度，显影时间长
  • 开窗偏小 → 曝光不足，显影不充分
  • 油墨脱落 → 前处理不当，固化温度低
  • 表面粗糙 → 涂布不均，异物污染

8. 表面处理 (Surface Finish)

8.1 HASL工艺 (Hot Air Solder Leveling)

工艺原理：

  • 板子浸入熔融焊料槽
  • 热风刀吹平多余焊料
  • 形成可焊性表面

工艺参数：

  • 助焊剂：松香型，比重1.08-1.12
  • 焊料温度：245-260°C
  • 浸入时间：2-4秒
  • 热风压力：10-30psi
  • 热风温度：350-450°C

厚度控制：

  • 焊料厚度：5-40μm
  • 均匀性要求：±50%
  • 表面平整度：<25μm

质量要求：

  • 可焊性：>95%润湿
  • 外观：光亮、无露铜
  • 厚度：1-40μm
  • 平整度：适合细间距器件

8.2 ENIG工艺 (Electroless Nickel Immersion Gold)

工艺流程：

  1. 清洁活化 → 2. 化学镍 → 3. 清洗 → 4. 化学金

化学镍工艺：

  • 镍盐浓度：4-6g/L
  • 还原剂：次磷酸钠
  • 工艺温度：82-88°C
  • pH值：4.4-4.8
  • 镍层厚度：3-6μm

化学金工艺：

  • 金盐浓度：1-2g/L
  • 还原剂：抗坏血酸
  • 工艺温度：88-92°C
  • pH值：4.4-4.8
  • 金层厚度：0.05-0.15μm

优势特点：

  • 表面平整度优异
  • 适合细间距BGA
  • 长期储存稳定
  • 多次回流焊接

质量控制：

  • 镍层厚度：3-6μm
  • 金层厚度：0.05-0.15μm
  • 可焊性：优异
  • 线材拉力：>1.0N

8.3 OSP工艺 (Organic Solderability Preservative)

工艺原理：

  • 有机保护膜涂覆
  • 防止铜表面氧化
  • 回流焊时挥发

工艺参数：

  • OSP液浓度：1-3%
  • 工艺温度：65-75°C
  • 浸泡时间：30-120秒
  • 膜厚：0.2-0.5μm

适用场景：

  • 成本敏感应用
  • 环保要求严格
  • 细间距元器件

8.4 其他表面处理

化学银 (Immersion Silver)：

  • 银层厚度：0.05-0.15μm
  • 导电性优异
  • 成本适中

化学锡 (Immersion Tin)：

  • 锡层厚度：0.8-1.2μm
  • 平整度好
  • 无铅环保

选择原则：

  • 元器件类型和间距
  • 焊接工艺要求
  • 成本预算考虑
  • 环保法规要求

9. 丝印 (Silkscreen)

9.1 丝印油墨类型

环氧型油墨：

  • 附着力强
  • 耐化学性好
  • 固化温度：150°C

丙烯酸型油墨：

  • 柔韧性好
  • 色彩鲜艳
  • UV固化可选

聚氨酯型油墨：

  • 耐磨性优异
  • 弹性好
  • 适合柔性板

9.2 丝印工艺参数

网版要求：

  • 网目数：200-400目/英寸
  • 丝径：20-30μm
  • 张力：20-25N/cm
  • 感光胶厚度：10-20μm

印刷参数：

  • 刮板硬度：70-85°邵氏A
  • 印刷压力：适中均匀
  • 印刷速度：50-200mm/s
  • 脱版距离：1-3mm

固化条件：

  • 固化温度：130-160°C
  • 固化时间：20-30分钟
  • 红外预烘：80-100°C

9.3 丝印内容要求

必要信息：

  • 元器件位号 (Reference Designator)
  • 极性标记 (Polarity Marks)
  • Pin 1标识
  • 测试点标识
  • 版本号、日期代码
  • 公司LOGO

设计规则：

  • 字符高度：≥0.8mm
  • 线宽：≥0.1mm
  • 字符间距：≥0.1mm
  • 避免与焊盘重叠

质量检验：

  • 字符清晰度
  • 位置准确性
  • 油墨附着力
  • 耐擦性测试

10. 成型 (Profiling)

10.1 数控铣床加工

设备要求：

  • 主轴转速：20,000-60,000 RPM
  • 定位精度：±0.02mm
  • 重复精度：±0.01mm
  • Z轴行程：≥50mm

刀具选择：

  • 直径：1-6mm
  • 材质：硬质合金
  • 刃数：2-4刃
  • 涂层：TiAlN, DLC

加工参数：

  • 切削速度：100-300m/min
  • 进给速度：1000-5000mm/min
  • 切削深度：0.5-2.0mm
  • 主轴转速：根据刀具直径调整

10.2 模具冲压

模具设计：

  • 冲压间隙：板厚的8-12%
  • 模具精度：±0.02mm
  • 刃口角度：0.5-2°
  • 模具寿命：>50万次

冲压参数：

  • 冲压力：根据板厚和硬度计算
  • 冲压速度：50-200次/分钟
  • 废料处理：真空吸附
  • 模具维护：定期研磨

10.3 激光切割

技术优势：

  • 无接触加工
  • 热影响区小
  • 切割精度高
  • 适合复杂形状

工艺参数：

  • 激光功率：根据板厚调整

3. 回流焊接 (Reflow Soldering)

3.1 温度曲线设计

四阶段温度曲线：

  1. 预热阶段：室温 → 150°C

    • 升温速率：1-3°C/s
    • 时间：60-120秒
    • 目的：蒸发溶剂，预热PCB

  2. 保温阶段：150°C → 183°C

    • 升温速率：0.5-1°C/s
    • 时间：60-120秒
    • 目的：激活助焊剂，去除氧化物

  3. 回流阶段：183°C → 230-250°C

    • 升温速率：1-3°C/s
    • 峰值温度：SAC305为230-250°C
    • 时间：>217°C保持40-90秒
    • 目的：形成IMC，完成焊接

  4. 冷却阶段：峰值温度 → 150°C

    • 降温速率：2-6°C/s
    • 自然冷却或强制冷却
    • 目的：凝固焊点，释放应力

3.2 回流炉设置

炉温分区：

  • 分区数量：5-12个温区
  • 温度控制：PID闭环控制
  • 温度精度：±2°C
  • 均匀性：±5°C

传送带系统：

  • 传送速度：0.5-3.0m/min
  • 载板角度：0-5°倾斜
  • 边轨宽度：可调节
  • 承载能力：根据板重量

气氛控制：

  • 氮气保护：氧含量<100ppm
  • 气流方向：对流循环
  • 压力控制：微正压
  • 温度均匀性：强制对流

3.3 焊接质量控制

焊点外观标准：

  • 焊料润湿角：<90°
  • 焊点形状：凹弧形
  • 表面光泽：金属光泽
  • 无气孔、裂缝、冷焊

焊点强度要求：

  • 拉力测试：符合IPC标准
  • 剪切强度：>5MPa
  • 疲劳寿命：>10^6次循环
  • 热循环：-40°C ~ +125°C

缺陷分析：

  • 冷焊：温度不足，时间不够
  • 焊料球：印刷偏移，温度过高
  • 立碑：加热不均，器件自对正
  • 润湿不良：助焊剂失效，氧化严重

3.4 温度曲线测试

测试工具：

  • 温度记录仪：多通道数据采集
  • 热电偶：K型或T型
  • 软件分析：曲线对比分析
  • 标准板：代表性测试载体

测试点选择：

  • 大器件焊点：BGA中心
  • 小器件焊点：0201电阻
  • PCB本体：不同厚度区域
  • 散热器件：大功率IC

曲线优化：

  • 升温速率：避免热冲击

  • 时间控制：防止过度焊接

  • 对位检查：器件位置偏移

7.3 图像分析技术

  • 密度分析：焊料均匀性

8. 清洗工艺 (Cleaning Process)

8.1 污染物类型

助焊剂残留：

  • 松香残留：活性化合物
  • 有机酸：腐蚀性物质
  • 卤素离子：Cl⁻, Br⁻, F⁻
  • 金属离子：Na⁺, K⁺等

制程污染：

  • 手印油污：操作员接触
  • 粉尘颗粒：环境污染
  • 氧化物：金属表面氧化
  • 静电吸附：微小颗粒

8.2 清洗方法

溶剂清洗：

  • 异丙醇 (IPA)：70-99%浓度
  • 清洗温度：室温-60°C
  • 超声波：40kHz频率
  • 清洗时间：3-10分钟

水基清洗：

  • 去离子水：电阻率>1MΩ·cm
  • 清洗温度：40-80°C
  • 表面活性剂：降低表面张力
  • 漂洗：多级逆流漂洗

等离子清洗：

  • 气体类型：氧气、氩气、氮气
  • 功率密度：0.1-1.0W/cm²
  • 处理时间：30-300秒
  • 真空度：10-100Pa

8.3 清洗设备

超声波清洗机：

  • 频率范围：25-80kHz
  • 功率密度：0.5-2.0W/cm³
  • 脱气功能：去除溶解气体
  • 温度控制：精确温控

喷淋清洗机：

  • 喷射压力：0.1-0.5MPa
  • 喷嘴设计：扇形、锥形
  • 传送速度：0.5-3.0m/min
  • 多级处理：清洗-漂洗-干燥

批式清洗机：

  • 装载方式：篮筐装载
  • 程序控制：自动化程序
  • 溶剂回收：蒸馏再生
  • 清洗容量：适合小批量

8.4 清洁度检测

离子污染度测试：

  • 测试方法：离子色谱法
  • 测试溶液：异丙醇/去离子水
  • 萃取条件：超声波，室温
  • 合格标准：<1.56μg NaCl当量/cm²

表面阻抗测试：

  • 测试电压：100V DC
  • 测试时间：60秒
  • 电极间距：标准间距
  • 合格标准：>10¹⁰Ω

目视检查：

  • 放大倍数：10-40倍
  • 检查项目：残留物、变色、腐蚀
  • 光照条件：充足均匀照明
  • 记录方式：照片记录

10. 三防漆涂覆 (Conformal Coating)

10.1 三防漆类型

丙烯酸树脂 (AR)：

  • 特点：透明、柔韧、易返修
  • 工作温度：-65°C ~ +125°C
  • 介电强度：>20kV/mm
  • 应用：一般环境保护

聚氨酯 (UR)：

  • 特点：耐磨、耐化学腐蚀
  • 工作温度：-65°C ~ +130°C
  • 硬度：较高
  • 应用：恶劣环境

有机硅 (SR)：

  • 特点：耐高温、柔韧性好
  • 工作温度：-65°C ~ +200°C
  • 耐湿性：优异
  • 应用：高温高湿环境

聚对二甲苯 (XY)：

  • 特点：致密性好、针孔少
  • 工艺：气相沉积
  • 厚度：5-50μm
  • 应用：高可靠性要求

10.2 涂覆工艺

刷涂工艺：

  • 适用：小批量、返修
  • 工具：毛刷、泡沫刷
  • 厚度控制：20-100μm
  • 优点：灵活、成本低

浸涂工艺：

  • 适用：大批量、简单PCB
  • 浸入速度：10-200mm/min
  • 提升速度：10-200mm/min
  • 厚度控制：粘度调节

喷涂工艺：

  • 喷枪类型：气动、静电
  • 喷涂压力：0.1-0.3MPa
  • 喷涂距离：100-300mm
  • 厚度控制：多道喷涂

选择性涂覆：

  • 设备：程控喷涂机
  • 精度：±0.1mm
  • 程序：CAD数据导入
  • 效率：高度自动化

10.3 遮蔽处理

遮蔽位置：

  • 连接器：防止涂覆污染
  • 开关按键：保证功能
  • 散热器：不影响散热
  • 测试点：便于测试

遮蔽方法：

  • 特殊涂层：水溶性涂层
